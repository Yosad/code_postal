J'ai réalisé un petit site permettant de demander à l'utilisateur de saisir :
- Un code postal français pour ensuite voir apparaitre sur une autre page la ville correspondante à ce code postal.
- Une ville française pour ensuite voir apparaitre sur une autre page le(s) code(s) postal(aux) correspondant(s) à cette ville. 

Je vous explique ma démarche :

J'ai commencé par créer un dossier Code_postal puis les différents fichiers index.php, index2.php, code_postal.php et ville.php
Les fichiers index.php et code_postal.php sont liés pour demander à l'utilisateur de saisir un code postal
Les fichiers index2.php et ville.php sont liés pour demander à l'utilisateur de saisir une ville

J'ai récupérer et insérer une base de données concernant les codes postaux de chaque ville sur ma base de données phpmyadmin.

Et à la fin je me suis occupé de design, qui reste encore un peu simpliste, avec le fichier bootstrap.css.
